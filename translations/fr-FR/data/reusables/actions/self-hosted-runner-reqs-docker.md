Les exécuteurs autohébergés doivent utiliser un système d’exploitation Linux et disposer de Docker pour exécuter les actions de conteneur Docker. Pour plus d’informations sur les impératifs des exécuteurs autohébergés, consultez « [À propos des exécuteurs autohébergés](/actions/hosting-your-own-runners/about-self-hosted-runners#requirements-for-self-hosted-runner-machines) ».
