GitHub APIには、[REST API](/rest)と[GraphQL API](/graphql)という2つの安定バージョンがあります。

## 非推奨のバージョン

### ベータ

ベータAPIは2014年4月22日に非推奨となりました。

### v2

API v2のサポートは2012年6月12日に廃止されました。

### v1

API v1のサポートは2012年6月12日に廃止されました。
