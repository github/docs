1. Opcionalmente, junto a "Uso este mes", haz clic en **Obtener informe de uso** para recibir un correo electrónico que contenga un vínculo para descargar un informe CSV de uso de almacenamiento de {% data variables.product.prodname_actions %}, {% data variables.product.prodname_registry %} y {% data variables.product.prodname_codespaces %}. El correo electrónico se envía a la dirección de correo electrónico principal de la cuenta. Puedes elegir si el informe debe abarcar los últimos 7, 30, 90 o 180 días.
   ![Descarga del informe CSV](/assets/images/help/billing/actions-packages-report-download.png)
