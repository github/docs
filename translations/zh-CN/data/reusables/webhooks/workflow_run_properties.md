| 键              | 类型    | 描述                                                                                       |
| -------------- | ----- | ---------------------------------------------------------------------------------------- |
| `action`       | `字符串` | 执行的操作内容. 可以是 `requested` 或 `completed` 之一。                                               |
| `workflow_run` | `对象`  | 工作流程运行。 包括 `artifacts_url`、`check_suite_id`、`conclusion`、`head_branch` 和 `head_sha` 等信息。 |
