| 键        | 类型    | 描述                                           |
| -------- | ----- | -------------------------------------------- |
| `action` | `字符串` | 执行的操作内容. 可以是以下选项之一：<ul><li> `added` - 用户接受加入仓库的邀请。</li><li>`removed` - 用户被删除仓库协作者角色。</li><li>`edited` - A user's collaborator permissions have changed. </li></ul> |
