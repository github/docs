### 关于赞助者个人资料

您的 {% data variables.product.prodname_sponsors %} 个人资料会告诉潜在的赞助者，他们为什么应该支持您。 人们点击您的个人资料或组织资料上的 **Sponsor（赞助）** 按钮，就会看到您的赞助个人资料。 我们建议个人资料中包括以下信息。

- 您参与的开源工作。
- 您为什么致力于开源开发。

您还可以设定目标来解释不同的赞助级别允许您为社区做些什么。
