## 关于 {% data variables.product.prodname_community_exchange %}

{% data reusables.education.about-github-community-exchange-intro %}

![Community Exchange 页面的屏幕截图](/assets/images/help/education/community-exchange-page.png)

你可以帮助你的同行学习开放源代码技能，成为项目维护者，并在经过验证的安全社区平台中开发你的 {% data variables.product.prodname_dotcom %} 项目组合和网络。

借助 {% data variables.product.prodname_community_exchange %}，你可以：
- 发现学生创建的存储库
- 用星号标记感兴趣的存储库
- 提交需要协作者的存储库
- 提交存储库以教授新技能
- 管理存储库的提交

有关详细信息，请参阅“[通过 GitHub Community Exchange 参与](/education/contribute-with-github-community-exchange)”。
