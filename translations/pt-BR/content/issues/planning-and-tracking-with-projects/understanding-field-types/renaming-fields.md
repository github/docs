{% data reusables.projects.project-settings %}
1. Clique no nome do campo que você deseja editar.
   ![Captura de tela mostrando um campo Iteração](/assets/images/help/projects-v2/select-single-select.png)
1. Em "Nome do campo", digite o novo nome para o campo.
   ![Captura de tela mostrando o nome do campo](/assets/images/help/projects-v2/field-rename.png)
1. Para salvar as alterações, pressione <kbd>Return/Enter</kbd>.
