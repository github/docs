Se você definir a permissão de metadados `No access` e selecionar uma permissão que exija acesso ao repositório, o GitHub substituirá sua seleção e definirá a permissão de metadados de novo para `Read-only`. Para definir a permissão de metadados como `No access`, primeiro, defina todas as permissões que exigem acesso ao repositório como `No access`.
