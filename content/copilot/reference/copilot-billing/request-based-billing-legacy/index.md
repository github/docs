> [!IMPORTANT] On June 1, 2026, {% data variables.product.github %} moved to usage-based billing. The articles in this category, which refer to {% data variables.product.github %}'s legacy premium request-based billing model, only apply to {% data variables.copilot.copilot_pro_short %} and {% data variables.copilot.copilot_pro_plus_short %} subscribers on an existing **annual plan** who remain on **request-based billing** after June 1, 2026.
