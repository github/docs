{% data variables.product.company_short %} offers several {% data variables.product.prodname_copilot %} plans designed for different types of users. Whether you're exploring AI-powered coding assistance for the first time, building software professionally, or managing Copilot across an organization, there's a plan tailored to your needs.

## Choose the right plan

### Individual plans

#### {% data variables.copilot.copilot_free %}

Get started with {% data variables.product.prodname_copilot_short %} at no cost. Ideal for developers who want to explore AI-assisted coding with access to a limited set of features and models.

#### {% data variables.copilot.copilot_student %}

Available to verified students. Includes unlimited code completions, an allowance of {% data variables.product.prodname_ai_credits %}, and limited access to chat and agent experiences.

#### {% data variables.copilot.copilot_pro %}

Designed for developers who use {% data variables.product.prodname_copilot_short %} regularly. Includes unlimited code completions, access to selected AI models, {% data variables.copilot.copilot_cloud_agent %}, and monthly AI credits.

Verified teachers and maintainers of popular open source projects may qualify for free access.

#### {% data variables.copilot.copilot_pro_plus %}

Includes everything in {% data variables.copilot.copilot_pro_short %}, plus access to premium AI models and a higher monthly AI credits allowance. Best suited for developers who rely heavily on AI-powered workflows.

#### {% data variables.copilot.copilot_max %}

Built for advanced and high-volume users. Includes everything in {% data variables.copilot.copilot_pro_plus_short %}, the highest individual AI credits allowance, and priority access to new models and features.

### Organization and enterprise plans

#### {% data variables.copilot.copilot_for_business %}

Designed for teams and organizations. Includes centralized policy management, a shared monthly AI credits pool, access to a broad model catalog, and {% data variables.copilot.copilot_cloud_agent %}.

#### {% data variables.copilot.copilot_enterprise %}

Includes all {% data variables.copilot.copilot_business_short %} capabilities, plus enterprise-focused features, larger AI credit allocations, and priority access to new models and capabilities.

> [!NOTE]
> {% data variables.product.prodname_copilot_short %} is currently unavailable for {% data variables.product.prodname_ghe_server %}.
